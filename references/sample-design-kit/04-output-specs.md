# 04 — Output Specs & Canva Canon

What to build, at what size, and **which existing design to clone**. The connected
Canva workspace has **no autofill brand templates**, so the workflow is: **clone a
proven design from the canon below, then swap in the brief's content** (see `05`).

---

## Dimensions by channel

### Presentation slides (16:9)
| Use | Size |
|---|---|
| Pitch deck / webinar / info session | **1920 × 1080 px** (16:9) |
- Title slide → numbered-column or Gaps→Bridge layouts → pillar/stat slides → CTA+QR closing.

### Social posts
| Platform / use | Size |
|---|---|
| Instagram / Facebook feed (square) | **1080 × 1080 px** |
| Instagram / Facebook portrait | **1080 × 1350 px** |
| Stories / Reels cover | **1080 × 1920 px** |
| LinkedIn / FB shared-link image | **1200 × 627 px** |
| Facebook event / link banner (wide) | **2048 × 1448 px** (existing "Register Post" size) |

### Recruitment / event graphics
| Use | Size |
|---|---|
| Poster (portrait, web) | **1080 × 1350** or **1240 × 1748** (≈A-ratio) |
| Webinar promo banner | **1920 × 1080** (16:9) or **1200 × 627** (link) |
| Countdown / announcement post | **1080 × 1080** |
| QR-code card | square, ≥ **1080 × 1080** (use the existing Cất Cánh QR design) |

### Docs & one-pagers
| Use | Size |
|---|---|
| One-pager / fact sheet / handout | **A4 (2480 × 3508 px @300dpi)** or **US Letter (2550 × 3300)** |
| Email header | **1200 × 400 px** |

> If a brief doesn't name a size, ask which channel — then pick from this table.
