# 03 — Voice, Copy & Naming

How Cất Cánh sounds — on slides, posts, and graphics. Marketing copy is **bilingual,
equal weight (Vietnamese + English).** Apply the same four standards the website copy
follows: **purposeful, concise, conversational, clear.**

---

## Voice

VISEMI / Cất Cánh sounds like a **trusted senior who believes in you**: expert but
warm, ambitious but honest, never salesy or stiff.

| Trait | Do | Don't |
|---|---|---|
| **Credible** | Concrete, specific, backed by the program structure. | Hype, superlatives with no substance. |
| **Aspirational** | "Take off." "Bridge the gap." Future-facing, hopeful. | Fearmongering or pity framing of students. |
| **Warm & human** | Second person ("you", "bạn"), real student stories. | Cold institutional jargon, passive voice. |
| **Clear** | Short lines, plain words, one idea per surface. | Dense paragraphs on a slide; acronyms unexplained. |

**Tone shifts by audience:** students → encouraging, motivating, peer-warm; donors/
partners → confident, impact- and ROI-focused, credible; experts/mentors → respectful,
purpose-driven. (See the `{AUDIENCE}` tag below.)

---

## Bilingual rules

- **Equal weight:** unless the brief says otherwise, produce both VI and EN. On a
  single surface, lead with the audience's language (students → VI lead; donors → EN
  lead) and place the other beneath, smaller or in a secondary weight.
- **Vietnamese keeps full diacritics — always.** `Cất Cánh`, `học bổng`, `bán dẫn`,
  `kỹ sư`. Never strip accents. Be Vietnam Pro renders them correctly.
- **Don't machine-mix** clumsy half-translations. Each language reads natively.
- **Numbers, currency, dates** are language-neutral but format per audience
  (e.g. `30/06/2026` for VI, `June 30, 2026` for EN). Pull all such values from the
  brief.

### Program name & key terms
- **Program name:** **Cất Cánh**. First English mention: **Cất Cánh (Takeoff)
  Fellowship**; afterwards "Cất Cánh" or "the Fellowship."
- **Org:** **VISEMI Foundation** (all-caps VISEMI). A U.S. 501(c)(3) non-profit.
- **Don't** write "Cat Canh" without diacritics in any final, user-facing copy.

## On-surface copy patterns (apply ux-writing rules)

- **Headlines:** 3–8 words, active, benefit-led. *"Take off into a global chip
  career." / "Cất cánh vào sự nghiệp bán dẫn toàn cầu."*
- **Eyebrow/kicker:** 1–2 words, UPPERCASE, green. `FELLOWSHIP` · `TUYỂN SINH` ·
  `WEBINAR`.
- **CTA buttons:** active imperative, 2–4 words. *Apply now / Đăng ký ngay* ·
  *Donate / Ủng hộ* · *Become a mentor / Trở thành cố vấn*. Never "Click here"/"Submit".
- **Support line:** one sentence, ≤ ~18 words, says the single most useful thing.
- **Stat labels:** noun phrase under a big number — *Fellows funded* / *Suất học bổng*.
- **One message per surface.** If you need two ideas, make two slides/posts.
- **Front-load** the important word; cut filler ("In order to" → "To").

## File / design naming convention

Match the existing Canva library so new work sorts alongside the canon. Pattern:

```
{LANG} | {AUDIENCE} | {TYPE}            ← decks, one-pagers, formal collateral
{TYPE} - {Descriptor} - {Size/Channel}  ← social & event graphics
```

- **`{LANG}`** = `VIE` · `ENG` (or `VIE+ENG` for a bilingual piece).
- **`{AUDIENCE}`** = `Uni` (students/universities) · `Donors` · `Partners` ·
  `Ambassador` · `Public`.
- **`{TYPE}`** = `Pitch Deck` · `One-pager` · `Webinar slide` · `Poster` ·
  `Register Post` · `Banner` · `Cover` · `QR codes`.

**Examples that already exist** (clone these names' style):
`ENG | Donors | Pitch Deck (5 slides)` · `VIE | Uni [official] | One-pager` ·
`Poster FB: Ambassador Recruit` · `Register Post - Square` · `Banner - Webinar Cất Cánh`.

Prefix drafts with `[Draft]` and append page count where useful (e.g. `(5 slides)`).
