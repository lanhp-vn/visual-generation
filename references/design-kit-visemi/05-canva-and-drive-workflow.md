# 05 — Canva & Google Drive Workflow

How to actually produce a design with the connected tools. Connectors are set up in
the Project UI (Canva + Google Drive). This file tells you how to use them on-brand.

---

## The golden rule

> **Layout from the canon. Content from the brief. Brand from the brand kit.**
> Never trust numbers, dates, or names baked into an existing design — many are
> outdated. Pull every fact from the user's brief; if it's missing, ask.

---

## Canva

### Brand kit
- Apply the **"Visemi" brand kit** (brand kit id `kAGeCJwpi8k`). A second kit exists
  (`kAGQJqkMeh0`, unnamed) — if "Visemi" doesn't carry the right colors/fonts/logos,
  ask the user which kit is authoritative before proceeding.
- The brand kit is the source of brand colors, Be Vietnam Pro, and the VISEMI logos
  inside Canva. The hexes/type in `02-brand-guidelines.md` must match it; if they ever
  diverge, flag it.

### Producing a design (no autofill templates exist)
1. **Find the right canon design** in `04` (or `search-designs` for a newer one).
2. **Clone it** (copy the design) so you don't overwrite the original.
3. **Replace content** with the brief's copy/numbers/images; apply the Visemi brand
   kit; keep the proven layout.
4. For brand-new formats with no canon: build from the component recipes in `02`
   and the layouts in `01`, on a fresh design at the size from `04`.
5. **Export / share:** give the user both the **edit** and **view** links, and offer
   a PNG/PDF export for the channel (PDF for decks/one-pagers, PNG for social).

### Canva capabilities available
- Search designs & brand templates, get design content/pages/thumbnails.
- Create from a brand template *if one is later added* (none today).
- Edit operations (text, images, elements), resize, export, comment.
- Upload assets from URL (e.g. a Drive image link or a logo from `assets/`).

### House rules for editing
- Keep within brand colors (`02`) and Be Vietnam Pro. Don't introduce new fonts/hues.
- Respect logo clear-space and pick the right logo variant for the background.
- One message per surface; generous white space; green used sparingly.
- Vietnamese text must retain diacritics after every edit — double-check.

## Google Drive

Use Drive to gather **inputs**, not to invent facts:
- **Copy & content:** pull approved text, talking points, or fact sheets the user
  points you to (search by title/`fullText`).
- **Photos & media:** fellow/cohort/event photos, headshots, partner logos.
- **Source files:** existing PPTX/PDF decks to reference or import into Canva.

**Workflow:** when the user references "the deck in Drive" or "our photos," search
Drive, confirm the right file with the user, then read/download and use it. Don't
pull numbers from a Drive doc into a design unless the user confirms they're current.

---

## Assets in this kit (fallback)
`assets/` holds source files you can attach/upload directly if the brand kit or Drive
doesn't have what you need:
- `logos/` — VISEMI logo, color + white, SVG + PNG, plus the square mark.
- `partner-logos/` — university/sponsor logos + `Sponsor List.csv` (partner reference).
- `reference/` — official `Colors.png` and `Typography.png` brand sheets.

---

## Quick checklist before delivering any design
- [ ] Right size for the channel (`04`)?
- [ ] Cloned from canon or built on real component recipes?
- [ ] Visemi brand kit applied; only brand colors + Be Vietnam Pro?
- [ ] All facts/numbers/dates came from the brief (none guessed/stale)?
- [ ] Bilingual handled correctly; Vietnamese diacritics intact?
- [ ] Correct logo variant; clear space respected?
- [ ] One clear message; green used as accent, not flood?
- [ ] Edit + view links (and an export) returned to the user?
